This widget will place a live football top assisters table in your website.

== Description ==

Add a football top assisters table to your WordPress website. To request a key sign up for your free trial at [statsfc.com](https://statsfc.com).

For a demo, check out [wp.statsfc.com/top-assisters](https://wp.statsfc.com/top-assisters/).

== Installation ==

1. Upload the `statsfc-top-assisters` folder and all files to the `/wp-content/plugins/` directory
2. Activate the widget through the 'Plugins' menu in WordPress
3. Drag the widget to the relevant sidebar on the 'Widgets' page in WordPress
4. Set the StatsFC key and any other options. If you don't have a key, sign up for a free trial at [statsfc.com](https://statsfc.com)

You can also use the `[statsfc-top-assisters]` shortcode, with the following options:

* `key` (required): Your StatsFC key
* `competition` (required*): Competition key, e.g., `EPL`
* `team` (required*): Team name, e.g., `Liverpool`
* `season` (optional): Season to show top assisters for, e.g., `2016/2017`
* `date` (optional): For a back-dated top assisters list, e.g., `2013-12-31`
* `highlight` (optional): Name of the team you want to highlight, e.g., `Liverpool`
* `limit` (optional): Maximum number of top assisters to show, e.g., `5`, `10`
* `show_badges` (optional): Display team badges, `true` or `false`
* `default_css` (optional): Use the default widget styles, `true` or `false`
* `omit_errors` (optional): Omit error messages, `true` or `false`

*Only one of `competition` or `team` is required.
